  2. 循环语句特征
for thing in list:
    thing *= 2
# 这种方式无法改变原始数据，

for i in range(len(list)):

  1. 字符串

    1. 切片操作(左闭右开)，可以使用切片操作实现字符串的翻转

       string[beg: end: step]

    2. join 是字符串的一个方法，用于将序列中的元素以指定的字符串作为分隔符连接起来，返回一个新的字符串。语法为：分隔符.join(序列) 其中，分隔符是一个字符串，用于在连接序列元素时插入到它们之间，序列可以是列表、元组等可迭代对象。

    4. str.find(substr, beg, end)

  5. python声明类

    1. 封装

      • Public 成员：没有任何访问限制。
      • Protected 成员：在成员名称前加上一个下划线 _，表示应该被视为受保护的
      • Private 成员：在成员名称前加上两个下划线 __，表示私有成员，只能在类内部访问。

    2. 继承

       class Animal:
     def sound(self):
         pass
         #在python之中可以用pass来实现占位
         #(python之中依靠缩进来分割逻辑)

 class Dog(Animal):
     def sound(self):
         print("Bark")

 class Cat(Animal):
     def sound(self):
         print("Meow")

    3. 多态

      Python 中的多态性可以通过子类重写父类函数

      多态性体现在对象的行为上，而不是在函数调用的绑定

      即使对象不是继承自同一个父类，只要有相同的方法，就可以被认为是多态的一种体现

      在 Python 中，多态性通常是通过“鸭子类型”（Duck Typing）来实现的。鸭子类型的理念是：“If it looks like a duck and quacks like a duck, it must be a duck.”。这意味着 Python 不会强制要求对象继承自特定的类或实现特定的接口，而是关注对象是否具有所需的方法和行为
